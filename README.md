# Personal Finance Dashboard - Frontend

## Description
This is the frontend client for the Personal Finance Dashboard. Built as a sleek Single Page Application (SPA), it enables users to seamlessly manage transactions, outline budgets, observe savings goals, track debts, and evaluate investment growth. It interfaces securely with the Node.js/Express backend APIs.

## Features
- **User Authentication:** Secure Login and Registration screens integrated with Context API.
- **Intelligent Dashboard:** Unified landing page containing summarized financial metrics.
- **Transactions Management:** Perform rapid CRUD operations spanning numerous categories.
- **Comprehensive Tracking:** Dedicated modules for Debts, Budgets, Savings Goals, and Investments.
- **Responsive Layout:** Tailored with Tailwind CSS and ShadCN components extending to mobile devices.

## Tech Stack
- React 19 / Vite
- React Router DOM
- Tailwind CSS
- Axios (for RESTful API requests)
- Recharts (for Dashboard visualization)
- Context API (for internal state)

## Installation Steps
1. Clone this repository (or navigate to the frontend folder).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your environment variables (e.g., in a `.env` file) as prompted by the backend configurations.
4. Start the development server:
   ```bash
   npm run dev
   ```
